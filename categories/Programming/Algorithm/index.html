자료구조

자료구조란?

  • 프로그램에서 사용할 많은 데이터를 메모리상에서 관리하는 여러 구현방법들이다.

  • 효율적인 자료구조가 성능 좋은 알고리즘의 기반이 된다.

  • 자료의 효율적인 관리는 프로그램의 수행속도와 밀접한 관련이 있다.

  • 여러 자료 구조 중에서 구현하려는 프로그램에 맞는 최적의 자료구조를 활용해야 하므로 자료구조에 대한 이해가 중요하다.
